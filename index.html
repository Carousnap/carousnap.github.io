Apa ini ?

Carousnap merupakan kumpulan foto yang di tampung kedalam satu wadah sehingga menghemat ruang tata letak dalam menampilkan banyak foto sekaligus.

Project ini dibuat dengan menggunakan Vanilla JavaScript dan CSS.


Dan ini bukanlah sebuah slideshow yang akan bergeser dalam rentang waktu tertentu. diperlukan aksi dari user untuk menggeser foto ke slide selanjutnya

Fitur

Number Slide

Jumlah Foto dan Urutan Foto yang sedang dilihat

Indicator Bar

Mempercepat Pengguna menuju foto yang ingin dilihat

Button Prev / Next

Pengguna dapat Menggeser 1 Foto / Slide Selanjutnya

Support Touch Event

diperangkat mobile hanya perlu Swipe untuk menggeser foto


Keunggulan Carousnap

Zero Dependencies

Tidak bergantung pada Package / Modul apapun, dibuat menggunakan Javascript dan CSS

Fit Cover / Position Center

Foto otomatis memiliki Posisi di tengah jika lebar sedikit melebihi pembungkusnya

Responsive

Lebar selalu Full / Memenuhi pembungkus jika kurang dari Max Width yang ditentukan

Support Browser

Hampir sebagian besar browser telah mendukung CSS Snap-Type

Easy to Use

Salin Code Carousnap , tempel ke project anda , dan Jalankan!

Fast

Ukuran fIle yang perlukan lebih kurang sebesar 14 KB

Instalasi

  • Saat ini instalasi hanya bisa dilakukan secara manual

  • Silahkan klik link berikut untuk download sumber file :

    Download disini !

  • Ekstrak sumber file dan pindahkan folder carousnap kedalam project anda!

  • Terdapat 2 File dalam Folder carousnap, sekarang tinggal menghubungkan script carousnap.js kedalam halaman utama project web anda Misal Index.html

  • Letakkan script berikut tepat diatas tag penutup body :

    <script type="text/javascript" src="./carousnap/carousnap.js" id="scriptCarousnap"></script>

  • Perlu diperhatikan, Jangan memisahkan file carousnap.css dan carousnap.js ! , pastikan file carousnap.css berada 1 folder dengan carousnap.js agar file css dapat terpanggil dengan baik.

Penggunaan

  • Mulai

    Salin Kode Berikut, Kode ini berfungsi untuk membuat Element CarouSnap. dan disinilah anda meletakkan beberapa foto anda

    <div class="carouSnap">
	<div class="numbSlide"></div>
	<div class="bnSlide"> </div>
	<div class="photoCollect">
		<!-- YOUR IMAGE HERE (Maximum 10 Photos & Minimum 1 Photo) -->
		<!-- <img src="#src" alt="#alt-image" title="#title-image" /> -->
	</div>
	<div class="indCat"></div>
</div>

    demi menghindari image stretch , pastikan dimensi perbandingan foto tidak terlalu jauh. misal 4:5 , 1:1

  • Resize

    Ukuran default max width adalah 600px, untuk mengubah ukuran tersebut, bisa dilakukan dengan menambahkan atribut data-width="value" dan data-height="value" pada element yang memiliki nama class carouSnap.

    <div class="carouSnap" data-width="value" data-height="value">
	<div class="numbSlide"></div>
	<div class="bnSlide"> </div>
	<div class="photoCollect">
		<!-- YOUR IMAGE HERE (Maximum 10 Photos & Minimum 1 Photo) -->
		<!-- <img src="#src" alt="#alt-image" title="#title-image" /> -->
	</div>
	<div class="indCat"></div>
</div>

  • Styling Rounded Border

    border melengkung pada setiap sisi cukup tambahkan .carouSnap-round pada class carouSnap

    <div class="carouSnap carouSnap-round">
	<div class="numbSlide"></div>
	<div class="bnSlide"> </div>
	<div class="photoCollect">
		<!-- YOUR IMAGE HERE (Maximum 10 Photos & Minimum 1 Photo) -->
		<!-- <img src="#src" alt="#alt-image" title="#title-image" /> -->
	</div>
	<div class="indCat"></div>
</div>

Untuk mendapatkan hasil ukuran yang baik, disarankan menggunakan tag pembungkus / parent element yang sudah di beri style ukuran tertentu.
